# Frontend

**Tecnologias**:
- Node.js 18 (LTS)
- Typescript
- NextJs

## Execução do projeto

1. Instalar requisitos:
   - Node/Npm v18 (lts) (recomendado instalar com o nvm)

2. Instalar dependências:
   ```
   npm install
   ```

3. Criar arquivo de env:
  ```
  cp .env.local .env.local.example
  ```

4. Inicializar o backend (leia o README do repo de backend)

5. Executar front:
   ```
   npm run dev
   ```
