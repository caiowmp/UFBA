Como as linguagens tinham que ser as criadas a partir da década de 1990, complicou um pouco minha situação por eu ter usado bastante C e C + +. Porém, percebi que Python, Java e JavaScript se encaixam perfeitamente aos requisitos.

A linguagem de programação Python foi criada por Guido van Rossum no ano de 1991. Ela foi influenciada por linguagens como: Haskell (compreensão de listas, funções anônimas e função map), ALGOL 68 e C (de onde é derivada parte da sintaxe). Dentre muitas aplicações que utilizam Python temos: compartilhador de arquivos Mnet, o sítio YouTube e o cliente original do BitTorrent. A experiência que eu tive em Python foi pela faculdade, na matéria Análise e Projeto de Algoritmos e na monitoria de Introdução à Lógica de Programação e eu gostei bastante por Python se tratar de uma linguagem muito fácil de se entender, enxuta (são precisas poucas linhas de código para fazer coisas gigantescas) e ser multiparadigma (ela é orientada a objetos, imperativa e funcional). Além disso, as aplicações de Python são bastante estáveis e o ecossistema é muito bom já que cada dia que passa ela se torna mais popular e mais requisitada.

A linguagem de programação Java foi criada por uma equipe de programadores chefiada por James Gosling na empresa Sun Microsystems no ano de 1995. Ela foi influenciada por linguagens como: C + +, Generic Java e Object Pascal. Dentre muitas aplicações que utilizam Java temos: o jogo mundialmente famoso Minecraft, Maestro (controlador de rovers em Marte) e Wikipedia Search. A experiência que eu tive em Java foi pela faculdade durante a matéria de programação orientada a objetos e eu gostei bastante por Java se tratar de uma linguagem orientada a objetos, paradigma que adorei utilizar. Ela é uma linguagem simples de se escrever e ler, suas aplicações são estáveis e o ecossistema, com a visão que eu tenho, hoje em dia está frio (poucas vagas de emprego e pouco interesse pela linguagem).

A linguagem de programação JavaScript foi criada por Brendan Eich no ano de 1995. Ela foi influenciada por linguagens como: C,  Java e Python. Dentre muitas aplicações que utilizam JavaScript temos: Skycatch, que usa o Node.js (um framework de JavaScript) para alimentar seus drones especializados que tiram fotos do canteiro de obras e as transforma em modelos 3D, o Google e a Microsoft utilizam o JavaScript para suas aplicações web, tais como sites de pesquisa. A experiência que eu tive em JavaScript foi pela faculdade durante a confecção de um trabalho na matéria de Sistemas Operacionais, onde fiz um escalonador de processos em uma página web e na implementação de web games a fim de desenvolver expertise na linguagem. Gostei bastante da linguagem, por ela ter essa relação com o front-end e ser multiparadigmas. Ela é uma linguagem um pouco complexa pra se ler, um pouco complexa de se escrever também (por conta das várias funções presentes no código, pelo fato de pode passar funções como parâmetros de outras funções), a confiabilidade dela é ótima pois suas aplicações são bastante estáveis e o ecossistema também é muito bom já que a comunidade de JavaScript é muito grande e engajada, além do mercado ser muito aquecido.
